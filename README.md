Caesar Cipher Encryption/Decryption Script

This Python script allows users to encrypt or decrypt text using a Caesar cipher. Users can input a text, choose between encryption and decryption, and specify a shift number between 1 and 25.

Features:
- Encrypt or decrypt text using a specified shift.
- Handle both lowercase and uppercase letters.
- Ignore spaces during transformation.
- Prompt user for retry in case of invalid input.


This script is a simple implementation of the Caesar cipher for educational and practical purposes.
